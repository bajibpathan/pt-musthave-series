Is the programming knowledge required for performance testing?
=============================================================
Programming Baiscs are mandatory because to customize performance test scripts.
	Variables and Data Types,

    Conditional Statements (if..else), 
    Loops (for, do..while, while), 
    Error handling and Debugging skills

What is Groovy?
================
Groovy is a Java-syntax-comptaible, object-oriented programming language that runs on the Java Virtual Machine (JVM)
It is a both static and dynamic langague

Features of Groovy
====================
- Flat learning curve
- Smooth Java integration
- Vibrant and rich ecosystem
- Supports both static and dynamic typing

Why do we need to learn Groovy?
===============================
- Groovy Recommended scripting language for Jmeter because it implements Compilable Interface.
- In Jmeter scripting will be done using JSR223 Elements.
